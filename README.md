# Ubuntu/Debian Installation

1. Clone the repository:
		
		git clone https://github.com/wownero/WowneroGUIWallet

2. Run installation script:

		cd /path/to/WowneroGUIWallet
		./install.sh

3. Start the wallet:
		
		python wallet.py
